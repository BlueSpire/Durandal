addBindingHandler

(
  • name
  • [config]
  • [initOptionsFactory]
)

Defined in src\durandal\js\composition.js:305

Registers a binding handler that will be invoked when the current composition transaction is complete.

Parameters:

  • name String

    The name of the binding handler.

  • [config] Object optional

    The binding handler instance. If none is provided, the name will be used to look up an existing handler which will then be converted to a composition handler.

  • [initOptionsFactory] Function optional

    If the registered binding needs to return options from its init call back to knockout, this function will server as a factory for those options. It will receive the same parameters that the init function does.
