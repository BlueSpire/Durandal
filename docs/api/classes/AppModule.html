Removes the callbacks for the specified events.

Parameters:

  • [events] String optional
    One or more events, separated by white space to turn off. If no events are specified, then the callbacks will be removed.
  • [callback] Function optional
    The callback function to remove. If callback is not provided, all callbacks for the specified events will be removed.
  • [context] Object optional
    The object that was used as this. Callbacks with this context will be removed.
