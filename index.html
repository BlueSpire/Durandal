The Future Is Here

Aurelia, The Next Generation of Durandal Is Here!

Aurelia has the best parts of yesterday's Durandal, re-imagined for the modern web. Prepare yourself for a framework that:

  • Natively understands ECMAScript 6 modules and classes.
  • Works well with Web Components.
  • Has an extensible HTML templating engine.
  • Provides simple two-way databinding to any object.
  • Has sensible conventions that make development effortless.
If you are starting a new project, targeting >= IE9 or modern browsers, you'll really want to use Aurelia.
